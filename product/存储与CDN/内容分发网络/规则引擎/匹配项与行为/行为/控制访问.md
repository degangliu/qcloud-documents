## 访问控制行为总览
| 分类-行为 | 场景            | 说明                                                         |
| --------- | --------------- | ------------------------------------------------------------ |
| 访问控制  | [IP 限制](#m1)  | 首页（url-index）：以根目录格式的请求URL可通过匹配           |
| 访问控制  | [防盗链](#m2)   | 全部文件（url-all） ：该域名下的所有请求都可通过匹配         |
| 访问控制  | [UA 限制](#m3)  | 文件后缀 （uri-file）：根据您的输入值匹配访问请求的后缀      |
| 访问控制  | [下行限速](#m4) | 文件目录（uri-directory）：根据您的输入值匹配访问请求的目录，输入值可为正则表达式 |
